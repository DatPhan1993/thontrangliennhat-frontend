# Thôn Trang Liên Nhất API - Docker

API cho website Thôn Trang Liên Nhất đã được đóng gói với Docker.

## Triển khai

### Với Docker Compose:
```bash
docker-compose up -d
```

### Cổng dịch vụ:
- API: Port 3000

## Thư mục quan trọng
- `/public/images`: Chứa hình ảnh hệ thống
- `/public/uploads`: Chứa tệp người dùng tải lên
